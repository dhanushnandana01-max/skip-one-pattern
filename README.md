# Skip One Pattern

A browser-based memory game inspired by the classic Simon Game with a unique twist. Instead of repeating the entire sequence, players must remember and repeat only the colors shown at the **odd positions (1st, 3rd, 5th, 7th...)**.

## 🎮 Live Demo

🔗 https://dhanushnandana01-max.github.io/skip-one-pattern/

---

## 📖 About the Game

Skip One Pattern is a memory game where each level adds a new color to the sequence.

Unlike the traditional Simon Game, players **ignore every alternate color** and only repeat the colors at **odd positions**.

## 🚀 How to Play

1. Press any key to start the game.
2. Watch the sequence carefully.
3. Remember only the **1st, 3rd, 5th, 7th...** colors.
4. Ignore the even-position colors.
5. Repeat the remembered colors in the correct order.
6. Continue until you make a mistake.
